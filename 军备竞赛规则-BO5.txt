⚡ 边狱巴士 · 军备竞赛模式 · 完整规则
===================================================================

本模式为独立赛制，与死斗模式平级。
采用BO5（五局三胜），核心为随机阵营抽选+升级制。

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
一、基础设定
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

• 赛制：BO5（五局三胜）
• 出战人数：4v4
• 不禁人
• 各阵营至少4人，不足4人者从散人池补足

阵营分级详见《阵营划分表_最终版.txt》。
五指仅作为败方支援出场，不纳入主阵营抽选。

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
二、阵营抽选机制
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

【开局抽选】
双方从下等池中各**随机抽取1个阵营**，公示后出战。
第一局双方均为下等。

【每局结束后的抽选】
  • 胜方：向上升一级 → 从上一级池中**随机抽取1个新阵营**
    下等 → 中等 → 上等（已达上等则维持本级重抽）

  • 负方：不升级 → 从本级池中**随机抽取1个新阵营**
    （双方可能抽到同一阵营，允许重复）



⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
三、换队机会（每方1次）
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

• 每方整个BO5有且仅有 **1次换队机会**
• 在抽到阵营后、公示前使用
• 使用时：**放弃当前抽到的阵营，从同级池中重新抽取1个**
• 换到的阵营必须接受，不可反悔
• 可在任意一局使用（包括第一局）
• 用掉即消失，不可保留

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
四、败方手指支援
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

【获得方式】
每输掉一局 → 立即从五指池中随机抽取1个指作为支援。
（公开抽取，增强节目效果）

【不重复规则】
若抽到已持有的手指则重抽，直至抽到新的指为止。
即不会持有重复的手指。

【叠加规则】
未使用的手指可无限累加。

【使用规则】
• 每局最多启用1个手指支援
• 同一局不能投入2个及以上手指
• 启用后该手指消耗消失
• 可在后续任意一局使用（不限使用时机）
• 未用完的手指可一直保留到BO5结束

【对战效果】
启用时：从该手指阵营中随机抽1人格，顶替出战阵营中
1个名额（4槽总数不变）。
该手指人格对方不可ban。

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
五、BO5 每局流程
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

① 抽选：双方从各自当前级别中**随机抽取1个阵营**
   （公开抽选，直播效果拉满）
② 公示：公布抽到的阵营
③ 换队（可选）：若该方还有换队机会，可在此步使用
④ 选人：
   - 先手方从最终确定的阵营中选4人，全部公示
   - 后手方看完对方阵容后，从最终阵营中选4人公示
   - 若本轮启用手指支援：从手指阵营中随机抽1人
     顶替1个名额（对手不可ban该人）
⑤ 4v4自动对战

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻
六、完整流程示例
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

────────────────────────────────────────
Game 1
  抽选：A抽到黑云会（下等）、B抽到流浪者（下等）
  公示 → 选人 → 对战
            ↓ A胜
  A升中等，B抽1指入袋
────────────────────────────────────────
Game 2
  抽选：A从中等池抽到散件事务所
        B从下等池抽到脑叶E.G.O·狂
  B换队机会 → 重抽到流浪者
  （换队机会消耗，不可再用）
  公示 → 选人（B启用手指）→ 对战
            ↓ B胜
  B升中等（指消耗），A抽1指入袋
────────────────────────────────────────
Game 3
  抽选：A从中等池抽到R公司
        B从中等池抽到山庄队
  公示 → 选人（A启用之前获得的指）→ 对战
            ↓ A胜
  A升上等（指消耗），B再抽1指入袋（累积2指）
────────────────────────────────────────
Game 4
  抽选：A从上等池抽到鸿园护院者
        B从中等池抽到五协会
  B启用1指 → 公示 → 选人 → 对战
            ↓ A胜
  A维持上等重抽，B再抽1指（累积3指）
────────────────────────────────────────
Game 5
  抽选：A从上等池重抽到脑叶E.G.O·悲
        B从中等池抽到六·十协会
  B手头3指，任选1个启用 → 最终决战
────────────────────────────────────────

七、与死斗模式的区别
⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

               死斗模式              │ 军备竞赛
─────────────────────────────────────┼─────────────────────────
赛制           BO3                   │ BO5
出战人数       原版设置                │ 4v4
禁人           有（分级禁人）          │ 不禁人
阵营获取       固定阵营池中自选        │ 每局随机抽取，赢家升级
升级机制       无                     │ 胜方升一级重新抽
                                      败方本级重新抽
换队机会       无                     │ 每方1次，可重抽本级阵营
手指规则       开赛抽1指，全程用1次    │ 开赛抽1指+每败1局抽1指
                                      可叠加，每局限用1个
